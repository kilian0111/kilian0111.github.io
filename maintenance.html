Maintenance

Contexte

Ce projet à été réalisé pour aider à la maintenance de locaux ; en effet, en période de covid les locaux sont vides. Ce projet permet aussi d'avoir un sytème informatique à la place d'utiliser des fiches de maintenance.

Objectif et Problématique

L'objectif de ce projet est de créer un formulaire avec les actions de maintenance effectuées. Pour cela, j'ai eu une fiche des actions à effectuer lors d'un contrôle de maintenance.

Les technologies

  • HTML / CSS
  • PHP
  • Bootstrap
  • javascript
  • jQuery
  • PostgreSQL
  • PGadmin

Gestion de projet

Pour réaliser ce projet j'ai utilisé un Raspberry pi comme serveur apache et de base de données. Ainsi ce projet est hébergé en local et disponible grâce à un nom de domaine chez OVH.
